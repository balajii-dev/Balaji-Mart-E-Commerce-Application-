# 🛒 Balaji Mart - Full Stack E-Commerce Application

A modern, high-performance E-commerce platform built with the **MERN Stack**. Balaji Mart provides a seamless shopping experience with a focus on responsive design, real-time data synchronization, and an intuitive user interface.

🚀 **Live Demo:** [balaji-mart-e-commerce-application.vercel.app](https://balaji-mart-e-commerce-application.vercel.app/)

---

## 📌 Key Features
* **Dynamic Product Catalog:** Browse products with real-time data fetched from a MongoDB database.
* **Integrated Cart System:** Add, remove, and update product quantities with persistent state.
* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop to prevent UI cut-off issues.
* **Secure API Communication:** Backend-to-frontend connectivity secured with custom CORS configurations.
* **Clean UI/UX:** Styled with Tailwind CSS for smooth transitions and professional aesthetics.

---

## 🛠️ Tech Stack
* **Frontend:** React.js (Hooks, Context API)
* **Backend:** Node.js & Express.js
* **Database:** MongoDB Atlas
* **Styling:** Tailwind CSS (Mobile-First approach)
* **Deployment:** Vercel (Frontend) & Render (Backend)

---



---

## 📱 Responsive Engineering
Following best practices for professional MERN developers, this project uses a fluid layout system. 
* **Adaptive Padding:** Replaced fixed left-padding (like `pl-20`) with responsive classes (`px-6 md:pl-20`) to ensure content fits perfectly on small screens.
* **Auto-Scaling Text:** Typography scales dynamically based on the viewport size to maintain readability across all devices.
